Todo
====

This is a simple todo app written in C.

Usage
-----

To create a new todo file or view an existing one:

    todo

To add a new task:

    todo buy milk
    todo "eat cookies"

To edit a task:

    todo -e 1 buy more milk

To move a task:

    todo -m 2 1

To mark a task:

    todo 1

To remove a task:

    todo -r 1
